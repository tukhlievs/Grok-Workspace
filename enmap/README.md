# Enmap

**Material You 3** introduction landing page for Enmap — AI agent that finds unique aesthetic photography and video locations based on your real-time location.

## Features
- Beautiful Google Material You 3 inspired design with dynamic geometric shapes
- Fully interactive demo with simulated AI search
- Smooth animations powered by Framer Motion
- Fixed, stable UI (no drift, no unwanted zooms)
- Responsive, production-ready Next.js 14 + TypeScript + Tailwind
- Russian language interface (optimized for Tashkent / Uzbekistan)

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS + custom Material You tokens
- Framer Motion (animations & modals)
- Lucide Icons
- Sonner (beautiful toasts)

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deployment
Ready for Vercel (recommended). Just connect the `enmap` folder or the whole repo.

## Design Philosophy
- Strong visual identity with geometric Google-style floating shapes
- Large rounded corners (3xl+), soft elevation shadows
- Vibrant yet harmonious primary color (#7c3aed)
- Inter font with Cyrillic support
- Fixed layout that never drifts on resize or interaction
- Maximum accessibility and performance
